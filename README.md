# 🔍 FakeBuster — ML-Powered Fake News Detection Web App

[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?logo=streamlit)](https://streamlit.io/)
[![Python](https://img.shields.io/badge/Python-3.10+-3776AB?logo=python)](https://www.python.org/)

> Interactive Streamlit web application for real-time fake news detection using Bi-LSTM, SVM, and Naive Bayes models with Doc2Vec embeddings.

## 🚀 Features

- **Web UI**: Streamlit app with model selector and calibration sidebar
- **4 Models**: Bi-LSTM (Best), Keras NN, SVM, Naive Bayes
- **Doc2Vec**: 300-dimensional text embeddings via gensim
- **Real-time**: Paste any news text and get instant classification

## 🏃 Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```
